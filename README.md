# Raycasting Engine
## Overview
This project is a template for implementing a basic raycasting engine in Python using Pygame. It serves as a foundation for anyone looking to understand the fundamentals of raycasting and how it can be used to simulate a 3D perspective in a 2D environment.

![swappy-20240808-225439](https://github.com/user-attachments/assets/85b3738d-20bc-4174-9362-cdfb6b254733)


## Prerequisites
* Python 3.x
    * Pygame `pip install pygame`
    * Sys
    * Math

## Installation
* Ensure Python 3.x is installed from python.org.
* Clone this repository or download and extract the ZIP file.

## Usage
```
python main.py
```

## Controls
* `W`: Move forward
* `S`: Move back
* `A`: Move left
* `D`: Move right
* `Left Arrow`: Rotate left
* `Right Arrow`: Rotate right
